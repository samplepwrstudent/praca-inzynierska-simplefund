

\section*{Streszczenie}
Niniejsza praca przedstawia projekt i implementację aplikacji webowej, przeznaczonej do monitorowania inwestycji oraz zarządzania portfelem indywidualnych użytkowników. System integruje informacje o różnych aktywach w przejrzystym interfejsie. Umożliwia rejestrację i logowanie, dodawanie operacji finansowych oraz wizualizację danych za pomocą wykresów i zestawień.

W pracy przeprowadzono analizę istniejących rozwiązań, przedstawiono architekturę systemu w postaci modelu C4 oraz opracowanie kluczowych funkcji, takich jak monitorowanie inwestycji, obliczanie wskaźników finansowych oraz wizualizację portfela. Zwrócono także uwagę na bezpieczeństwo danych, przetestowano wydajność i użyteczność aplikacji. Efektem jest w pełni funkcjonalne narzędzie wspierające zarządzanie zróżnicowanym portfelem inwestycyjnym.

\section*{Abstract}
This thesis presents the design and implementation of a web application intended for monitoring investments and managing individual users’ portfolios. The system integrates information on various assets within a clear and intuitive interface. It enables user registration and authentication, the addition of financial operations, as well as data visualisation through charts and summaries.

The thesis includes an analysis of existing solutions, a presentation of the system architecture using the C4 model, and a description of the implementation of key functionalities such as investment monitoring, calculation of financial indicators, and portfolio visualisation. Attention is also given to data security, alongside performance and usability testing of the application. The result is a fully functional tool that supports the management of a diversified investment portfolio.
