

\section*{Streszczenie}
Niniejsza praca przedstawia projekt i implementację aplikacji webowej, przeznaczonej do monitorowania inwestycji oraz zarządzania portfelem indywidualnych użytkowników. System integruje informacje o różnych aktywach w przejrzystym interfejsie. Umożliwia rejestrację i logowanie, dodawanie operacji finansowych oraz wizualizację danych za pomocą wykresów i zestawień.

Aplikacja została zrealizowana z wykorzystaniem React 18, TypeScript, Material UI 7, Firebase Authentication, Firestore oraz Flask API hostowanego na platformie Heroku. W pracy przeprowadzono analizę istniejących rozwiązań, przedstawiono architekturę systemu w postaci modelu C4 oraz opracowano najważniejsze funkcje, takie jak monitorowanie inwestycji, obliczanie wskaźników finansowych oraz wizualizację portfela. Zwrócono także uwagę na bezpieczeństwo danych, przetestowano poprawne działanie aplikacji za pomocą Cypress.

Testy wydajnościowe przeprowadzono za pomocą Google Lighthouse. Analiza wskaźników Web Vitals wykazała wyniki: 72/100 dla wydajności, 91/100 dla dostępności, 100/100 dla najlepszych praktyk oraz 92/100 dla SEO. Efektem jest w pełni funkcjonalne narzędzie wspierające zarządzanie zróżnicowanym portfelem inwestycyjnym.

\section*{Abstract}
This thesis presents the design and implementation of a web application for monitoring investments and managing individual user portfolios. The system integrates information about various assets in a transparent interface. It allows users to register and log in, add financial transactions, and visualize data using charts and reports.

The application was developed using React 18, TypeScript, Material UI 7, Firebase Authentication, Firestore, and Flask API hosted on the Heroku platform. The thesis analyzes existing solutions, presents the system architecture in the form of a C4 model, and develops key features such as investment monitoring, financial indicator calculation, and portfolio visualization. Data security was also taken into account, and the correct operation of the application was tested using Cypress.

Performance tests were conducted using Google Lighthouse. The analysis of Web Vitals indicators showed the following results: 72/100 for performance, 91/100 for accessibility, 100/100 for best practices, and 92/100 for SEO. The result is a fully functional tool that supports the management of a diversified investment portfolio.
